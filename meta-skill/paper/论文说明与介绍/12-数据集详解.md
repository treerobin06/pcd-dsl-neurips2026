## 各数据集详解

### 1. Flight 偏好学习数据集

**来源**: Qiu et al., 2026 (Nature Communications) 的 Zenodo 数据集

**任务描述**:
- 用户有隐藏的航班偏好（4 个特征的线性权重）
- 每轮从 3 个航班中选择一个
- LLM 需要推断用户偏好并推荐最佳航班
- 共 5 轮交互

**数据格式** (`flight.jsonl`):
```json
{
  "idx": 0,
  "reward_fn": [1.0, -0.5, 0.0, 0.5],    // 隐藏的偏好权重
  "features": ["departure_time", "duration", "number_of_stops", "price"],
  "rounds": [
    {
      "options": [
        {"departure_time": "02:00 PM", "duration": "4h24m", "price": "$370", ...},
        ...
      ],
      "chosen": 2  // 用户选了第 3 个
    },
    ...
  ],
  "rounds_numpy": [...]  // 数值化版本
}
```

**完全覆盖设计**: 624 = $5^4 - 1$
- 每个特征的偏好权重取值: {-1, -0.5, 0, 0.5, 1}
- 去掉全零向量 (0,0,0,0)
- 覆盖所有可能的偏好组合

**变体数据集** (用于 LOO 泛化):
- Flight-2F: 2 个特征, 24 个样本
- Flight-3F: 3 个特征, 124 个样本
- Flight-5F: 5 个特征, 3124 个样本
- Flight-6F: 6 个特征, 15624 个样本

**在论文中的使用方式**:
- PCD 诊断: 200 个样本 × 6 个模型
- 23 策略消融: 全部 624 个样本 × GPT-4o-mini
- 端到端验证: 全部 624 个样本
- DSL 等价性: 250 个样本

### 2. Hotel 偏好学习数据集

**来源**: 同 Flight

**任务描述**: 与 Flight 类似，但是酒店推荐
- 4 个特征: price, distance, rating, cleanliness
- 124 个样本

**在论文中的使用方式**:
- 仅用于 LOO 泛化测试（归纳器在 Flight 上训练，测试能否泛化到 Hotel）

### 3. TextBandit 多臂赌博机

**来源**: Lim et al., 2025

**任务描述**:
- 多臂赌博机 (Multi-Armed Bandit) 问题
- 每个臂有未知的成功概率
- LLM 需要通过试探来学习最优策略

**规模**: 4 种配置（臂数/先验的不同组合）

**推理家族**: 共轭更新 (Beta-Bernoulli)
- 先验: Beta(α, β)
- 观测 reward 后更新: Beta(α+s, β+f)

**在论文中的使用方式**:
- DSL 等价性验证: 50 个实例

### 4. BLInD (Bayesian network Language Inference Dataset)

**来源**: Nafar et al., 2025

**任务描述**:
- 合成贝叶斯网络的精确推断
- 自然语言描述 → P(query_var = val | evidence)

**规模和结构**:
- 900 个问题实例
- 网络深度: 2-10（每个深度 100 个）
- 所有变量为二元（取值 0 或 1）
- 树状结构（每个节点最多一个父节点）

**输入格式**: 自然语言描述，包含：
- 变量名和取值
- 因果关系描述（"X causes Y with probability p"）
- 查询和证据

**在论文中的使用方式**:
- PCD 诊断: 全部 900 个样本 × 多模型
- PAL 对比: 全部 900 个样本
- Compile-time 对比: 全部 900 个样本
- DSL 等价性: 全部 900 个样本
- 成本核算: 基于 900 个查询

### 5. bnlearn 标准网络

**来源**: Scutari, 2010 (bnlearn R package)

**四个网络**:

| 网络 | 节点 | 边 | 特点 |
|------|------|-----|------|
| **Asia** | 8 | 8 | 肺病诊断，二元变量 |
| **Child** | 20 | 25 | 儿科诊断，多值变量（最多 6 值） |
| **Insurance** | 27 | 52 | 汽车保险，多值变量（最多 5 值） |
| **Alarm** | 37 | 46 | 重症监护报警，多值变量（最多 4 值） |

**查询生成**:
- 每个网络随机生成 30 个推断查询
- 随机选择查询变量、证据变量及取值
- Gold answer 用 pgmpy 计算

**在论文中的使用方式**:
- 扩展性测试: 120 个查询
- PCD 验证: Compute|GoldParse = 0%

### 6. Naive Bayes 合成数据集

**来源**: 论文自己合成（生成脚本将发布）

**任务描述**:
- 医学诊断任务
- 给定症状集合，推断最可能的疾病

**数据规模和参数**:
- 200 个问题实例
- 3-6 种疾病
- 4-8 个症状
- 疾病先验和条件概率随机生成

**在论文中的使用方式**:
- Held-out 泛化测试
- PCD 诊断验证

### 7. HMM Forward 合成数据集

**来源**: 论文自己合成（生成脚本将发布）

**任务描述**:
- 隐马尔可夫模型前向滤波
- 给定观测序列，推断最后一个时间步的隐状态分布

**数据规模和参数**:
- 100 个问题实例
- 3-5 个隐状态
- 4-8 个观测时间步
- 转移概率和发射概率随机生成

**在论文中的使用方式**:
- Held-out 泛化测试
- PCD 诊断验证

### 数据分割原则

- **归纳样本**: 1 个及以上（k=1 已验证可行，论文实验中用 3-5 个）
- **验证样本**: 与归纳样本不重叠（用于 Gate 2 验证）
- **测试样本**: 与以上都不重叠（用于最终评估）
